# Markdown 语法 10分钟入门指南

> Markdown 是一种轻量级标记语言，用简洁的符号就能排版出美观的文档，广泛应用于 README、博客、笔记等场景。

---

## 一、标题与段落（2分钟）

### 标题
用 `#` 符号表示标题，1-6 个 `#` 对应 H1-H6：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 段落与换行
- 段落之间用**空行**分隔
- 行末加**两个空格**实现换行

### 实际效果
# 一级标题
## 二级标题

这是第一段。

这是第二段。

## 三、列表（2分钟）

### 无序列表
用 `-`、`+` 或 `*` 开头：

```markdown
- 苹果
- 香蕉
- 橙子
```

### 有序列表
用数字加 `.` 开头：

```markdown
1. 第一步
2. 第二步
3. 第三步
```

### 嵌套列表
子列表前加缩进（2 个空格或 1 个 Tab）：

```markdown
- 水果
  - 苹果
  - 香蕉
- 蔬菜
  - 胡萝卜
```

### 实际效果

- 水果
  - 苹果
  - 香蕉
- 蔬菜
  - 胡萝卜

## 五、代码与引用（2分钟）

### 行内代码
用反引号 `` ` `` 包裹：

```markdown
使用 `print()` 函数输出内容
```

### 代码块
用三个反引号 ``` 包裹，可指定语言：

````markdown
```python
def hello():
    print("Hello World")
```
````

### 引用
用 `>` 开头：

```markdown
> 这是一段引用文字。
> 可以跨越多行。
```

### 实际效果

使用 `print()` 函数输出内容

```python
def hello():
    print("Hello World")
```

> 这是一段引用文字。
> 可以跨越多行。

# 测试题

## 单选题

### 第1题
在 Markdown 中，如何创建二级标题？

A. `* 二级标题`
B. `## 二级标题`
C. `// 二级标题`
D. `<h2>二级标题</h2>`

<details>
<summary>点击查看答案</summary>

**答案：B**

解析：Markdown 中使用 `#` 符号表示标题，数量对应标题级别。`##` 表示二级标题。A 是无序列表语法，C 是注释语法，D 是 HTML 标签写法。

</details>

### 第2题
以下哪个语法可以将文字显示为**粗体**？

A. `*文字*`
B. `_文字_`
C. `**文字**`
D. `~~文字~~`

<details>
<summary>点击查看答案</summary>

**答案：C**

解析：`**文字**` 或 `__文字__` 可以创建粗体。`*文字*` 和 `_文字_` 创建斜体，`~~文字~~` 创建删除线。

</details>

### 第3题
创建一个指向 `https://example.com`、显示文字为"示例"的链接，正确语法是？

A. `[https://example.com](示例)`
B. `[示例](https://example.com)`
C. `(示例)[https://example.com]`
D. `链接: https://example.com`

<details>
<summary>点击查看答案</summary>

**答案：B**

解析：Markdown 链接语法是 `[显示文字](URL)`。中括号内是显示文字，小括号内是目标地址。

</details>

### 第4题
在 Markdown 中插入代码块，应该使用什么符号？

A. 单引号 `'`
B. 双引号 `"`
C. 三个反引号 `\`\`\``
D. 井号 `#`

<details>
<summary>点击查看答案</summary>

**答案：C**

解析：代码块使用三个反引号（```）包裹，可以在开头指定语言以获得语法高亮。行内代码使用单个反引号。

</details>

### 第5题
以下哪个符号用于创建无序列表？

A. `>`
B. `#`
C. `-` 或 `*`
D. `---`

<details>
<summary>点击查看答案</summary>

**答案：C**

解析：`-`、`*` 或 `+` 都可以创建无序列表。`>` 用于引用，`#` 用于标题，`---` 用于创建分割线。

</details>

## 判断题

### 第6题
Markdown 中，`*文字*` 和 `_文字_` 都可以创建斜体。

<details>
<summary>点击查看答案</summary>

**答案：✓ 正确**

解析：`*文字*` 和 `_文字_` 确实都可以创建斜体效果，两者完全等价，可以根据个人喜好选择。

</details>

### 第7题
Markdown 的图片语法是 `(图片描述)[图片地址]`。

<details>
<summary>点击查看答案</summary>

**答案：✗ 错误**

解析：正确的图片语法是 `![图片描述](图片地址)`，以感叹号 `!` 开头，不是小括号开头。这是链接和图片的主要区别。

</details>

### 第8题
在 Markdown 中，两个段落之间需要用空行分隔。

<details>
<summary>点击查看答案</summary>

**答案：✓ 正确**

解析：正确。Markdown 中段落由空行分隔，如果没有空行，相邻的文本会被视为同一段落。

</details>

### 第9题
`~~删除线~~` 可以创建带删除线的文字效果。

<details>
<summary>点击查看答案</summary>

**答案：✓ 正确**

解析：正确。用两个波浪线 `~~` 包裹文字可以创建删除线效果，显示为 ~~删除线~~。

</details>

### 第10题
Markdown 原生支持所有 HTML 标签。

<details>
<summary>点击查看答案</summary>

**答案：✗ 错误**

解析：错误。虽然很多 Markdown 解析器允许内嵌部分 HTML 标签，但这不属于 Markdown 的原生语法。纯净的 Markdown 只包含其定义的标记符号，不同平台对 HTML 的支持程度也不尽相同。

</details>
