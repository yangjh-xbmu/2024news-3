## 核心知识点

### 变量是什么

变量是数据的"容器"，给数据贴上标签。

```javascript
let name = "小明"   // name 指向 "小明"
let age = 18        // age 指向数字 18
```

### 三种声明方式

| 方式 | 重新赋值 | 重复声明 | 使用场景 |
|------|---------|---------|---------|
| `var` | 能 | 能 | 不用了（历史遗留） |
| `let` | 能 | 不能 | 需要变化的值 |
| `const` | 不能 | 不能 | 不变的值 |

```javascript
let count = 1
count = 2        // 可以改变

const PI = 3.14
PI = 3.15        // 报错！常量不能改
```

### 一句话总结

> 变量是数据的标签，`let` 给会变的量，`const` 给不变的量。
